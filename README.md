# HamClock Standards

This repository serves as the official specification of the HamClock Standards, specifically comprising:

* [HamClock Client to Backend API](./api-backend/)
* [Proposals](./proposals/) - additions or variations that have been proposed but not yet adopted
* and the governing [LICENSE](./LICENSE.md)

The purpose of the HamClock Standards Specifications is to encourage
full interoperability between the many HamClock backend and client implementations,
with the objective of ensuring that any compatible client implementation may be used with any compatible backend
implementation, including the original Clear Sky Institute services (while they are still available).
